# Commands

1. Prepare virtual environment

    ``` shell
    python3.12 -m venv venv

    source venv/bin/activate
    ```

2. Install dependencies

    ``` shell
    pip3 install                    \
    --requirement requirements.txt
    ```

3. Start JupyterLab

    ``` shell
    jupyter-lab \
    --no-browser
    ```
