# Commands

- Prepare virtual environment

    ```bash
    # PyTorch does not support Python 3.13 yet
    python3.12 -m venv venv
    
    source venv/bin/activate
    pip install -r ./requirements.txt
    ```

- Start JupyterLab

    ```bash
    jupyter-lab
    ```
